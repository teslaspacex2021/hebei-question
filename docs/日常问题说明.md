# 日常问题说明

日常问题用于记录和处理日常运营中的单条问题，每次发起仅可填报一条。根据发起层级与办理层级不同，分为三种流程。

---

## 一、表单字段说明

新建/编辑日常问题时，表单采用竖向布局，字段如下：

| 字段名称 | 是否必填 | 说明 |
| -------- | -------- | ---- |
| 问题内容 | 是 | 描述日常问题的具体内容，支持多行文本 |
| 主要答复部门 | 是 | 负责牵头答复与办理的主办部门 |
| 配合部门 | 否 | 协助办理的配合部门，支持多选 |
| 期望答复时间 | 是 | 期望获得答复或完成办理的截止日期 |
| 备注 | 否 | 补充说明、背景信息等 |

**填写规则：**

- 每次发起仅可填报 **1 条** 日常问题
- 不含调研批次标题、问题状态、问题分类等调研问题专属字段
- 不含附件字段
- 提交后进入对应流程的首个审批环节，基本信息不可再编辑

## 二、流程分类

| 流程类型 | 发起方 | 办理方 | 适用场景 |
| -------- | ------ | ------ | -------- |
| 省公司流程 | 省公司 | 省公司 | 省公司内部发起并在省公司内部办理 |
| 市公司内部流程 | 市公司 | 市公司 | 市公司内部发起并在市公司内部办理 |
| 市→省跨级流程 | 市公司 | 省公司 | 市公司发起，需升级至省公司办理 |

## 三、省公司流程

省公司发起、省公司内部办理。

| 节点 | 环节 | 说明 |
| ---- | ---- | ---- |
| 1 | 省公司发起人 | 发起日常问题并填写表单 |
| 2 | 省公司发起人部门主任 | 审核发起人提交的问题 |
| 3 | 省公司接单部门主任 | 接单部门主任确认接单 |
| 4 | 省公司一级办理（可跳过） | 一级办理人处理，该环节可跳过 |
| 5 | 省公司二级办理 | 二级办理人具体落实办理 |
| 6 | 省公司接单部门主任 | 接单部门主任审核办理结果 |
| 7 | 省公司发起人 | 发起人确认办理结果 |
| — | 问题提出人评价 | 问题提出人对办理情况进行评价 |
| — | 发起人确认 | 发起人最终确认 |
| — | 结束 | 流程办结 |

### 3.1 省公司流程 — 环节路由说明

> 办理人需在详情页选择办理路由、填写办理意见后点击「提交办理」。一级办理环节支持跳过。

| 节点 | 当前环节 | 处理人 | 可选路由 | 目标环节 | 路由类型 | 说明 |
| ---- | -------- | ------ | -------- | -------- | -------- | ---- |
| 1 | 省公司发起人 | 发起人 | 提交办理 | 省公司发起人部门主任 | 前进 | 填写表单后提交，进入部门主任审核 |
| 2 | 省公司发起人部门主任 | 发起人部门主任 | 退回发起人 | 省公司发起人 | 退回 | 材料不完整或有误，退回修改 |
| 2 | 省公司发起人部门主任 | 发起人部门主任 | 提交至接单部门主任 | 省公司接单部门主任 | 前进 | 审核通过，转主办部门接单 |
| 3 | 省公司接单部门主任（接单） | 接单部门主任 | 退回发起人部门主任 | 省公司发起人部门主任 | 退回 | 接单部门有误或需重新分派 |
| 3 | 省公司接单部门主任（接单） | 接单部门主任 | 提交至一级办理 | 省公司一级办理 | 前进 | 确认接单，进入一级办理 |
| 4 | 省公司一级办理 | 一级办理人 | 跳过一级办理 | 省公司二级办理 | 跳过 | 无需一级办理，直接进入二级 |
| 4 | 省公司一级办理 | 一级办理人 | 提交至二级办理 | 省公司二级办理 | 前进 | 一级办理完成，提交二级落实 |
| 5 | 省公司二级办理 | 二级办理人 | 提交至接单部门主任 | 省公司接单部门主任（审核） | 前进 | 办理完成，提交部门主任审核；**可更新办理进展** |
| 6 | 省公司接单部门主任（审核） | 接单部门主任 | 退回二级办理 | 省公司二级办理 | 退回 | 办理结果不达标，退回补充 |
| 6 | 省公司接单部门主任（审核） | 接单部门主任 | 提交至发起人 | 省公司发起人 | 前进 | 审核通过，提交发起人确认 |
| 7 | 省公司发起人（确认） | 发起人 | 提交至问题提出人评价 | 问题提出人评价 | 前进 | 发起人认可办理结果 |
| — | 问题提出人评价 | 问题提出人 | 提交至发起人确认 | 发起人确认 | 前进 | 完成满意度/结果评价 |
| — | 发起人确认 | 发起人 | 结束 | 结束 | 结束 | 最终确认，流程办结 |

## 五、市→省跨级流程

市公司发起、升级至省公司办理。与市公司内部流程的主要差异在于：**第 3 步需经市公司分管副总审批后，转交省公司办理**。

| 节点 | 环节 | 说明 |
| ---- | ---- | ---- |
| 1 | 市公司发起人 | 发起日常问题并填写表单 |
| 2 | 市公司发起人部门主任 | 审核发起人提交的问题 |
| 3 | **市公司分管副总** | 跨级升级审批，同意后转交省公司办理 |
| 4 | 省公司接单部门主任 | 省公司接单部门主任确认接单 |
| 5 | 省公司一级办理（可跳过） | 省公司一级办理人处理，该环节可跳过 |
| 6 | 省公司二级办理 | 省公司二级办理人具体落实办理 |
| 7 | 省公司接单部门主任 | 省公司接单部门主任审核办理结果 |
| 8 | 市公司发起人 | 市公司发起人确认办理结果 |
| — | 问题提出人评价 | 问题提出人对办理情况进行评价 |
| — | 发起人确认 | 发起人最终确认 |
| — | 结束 | 流程办结 |

## 六、路由规则说明

### 6.1 路由类型

| 路由类型 | 含义 | 典型场景 |
| -------- | ---- | -------- |
| 前进 | 流转至下一办理环节 | 审核通过、办理完成 |
| 退回 | 退回到前一或指定环节 | 材料有误、办理不达标 |
| 跳过 | 跳过当前环节 | 一级办理无需执行时直接进入二级 |
| 结束 | 流程办结 | 发起人最终确认后关闭 |

### 6.2 办理要求

| 要求项 | 说明 |
| ------ | ---- |
| 办理路由 | 存在多个路由时，须先选择路由再提交 |
| 办理意见 | 每次提交办理须填写办理意见 |
| 进展更新 | 一级办理、二级办理环节支持通过进度条更新办理进展 |
| 处理人显示 | 选择路由后，系统展示下一步处理人姓名与部门 |

### 6.3 三种流程路由差异对照

| 环节位置 | 省公司流程 | 市公司内部流程 | 市→省跨级流程 |
| -------- | ---------- | -------------- | ------------- |
| 第 2 步后 | → 省公司接单部门主任 | → 市公司接单部门主任 | → **市公司分管副总** |
| 第 3 步后 | → 省公司一级办理 | → 市公司一级办理 | → 省公司接单部门主任 |
| 审核后确认人 | 省公司发起人 | 市公司发起人 | 市公司发起人 |
| 退回起点 | 省公司内部各环节互退 | 市公司内部各环节互退 | 可退回至市公司或省公司对应环节 |

## 七、三种流程对比

| 节点 | 省公司流程 | 市公司内部流程 | 市→省跨级流程 |
| ---- | ---------- | -------------- | ------------- |
| 1 | 省公司发起人 | 市公司发起人 | 市公司发起人 |
| 2 | 省公司发起人部门主任 | 市公司发起人部门主任 | 市公司发起人部门主任 |
| 3 | 省公司接单部门主任 | 市公司接单部门主任 | **市公司分管副总** |
| 4 | 省公司一级办理（可跳过） | 市公司一级办理（可跳过） | 省公司接单部门主任 |
| 5 | 省公司二级办理 | 市公司二级办理 | 省公司一级办理（可跳过） |
| 6 | 省公司接单部门主任 | 市公司接单部门主任 | 省公司二级办理 |
| 7 | 省公司发起人 | 市公司发起人 | 省公司接单部门主任 |
| 8 | — | — | 市公司发起人 |
| 收尾 | 问题提出人评价 → 发起人确认 → 结束 | 问题提出人评价 → 发起人确认 → 结束 | 问题提出人评价 → 发起人确认 → 结束 |

**关键差异：**

- **省公司流程**：全程在省公司内部流转
- **市公司内部流程**：全程在市公司内部流转，第 3 步为市公司接单部门主任
- **市→省跨级流程**：第 3 步为市公司分管副总审批，之后由省公司承接办理，最后回到市公司发起人确认

## 八、与调研问题的区别

| 对比项 | 日常问题 | 调研问题 |
| ------ | -------- | -------- |
| 填报数量 | 每次 1 条 | 支持多条 |
| 调研信息 | 无 | 需填写调研批次、时间、地点等 |
| 问题分类 | 无 | 有 |
| 问题状态 | 无 | 有（已解决/未解决） |
| 地市问题确认 | 无 | 有 |
| 地市区县评价 | 无 | 有 |
| 附件 | 无 | 支持 |
| 流程分支 | 按发起/办理层级分三种 | 统一调研流程 |
